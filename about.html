About KidsBank

This is what we do

This is a banking program for children. Children can use this site so that they learn to save and manage money. Users register an account to use the site. If a user sets a goal to save money, enter it on the site.

A number of goals can be determined for each objective, and the target amount and current savings can be tracked to show the progress status and the expected date of completion. The user can enter a transaction that uses money, and it is stored on the site and displayed. The Statistic section provides a visual summary of the entire content.
